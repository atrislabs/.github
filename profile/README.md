## Atris Labs

We build **Atris**, an AI operating system for businesses.

Atris runs dedicated AI agents inside one isolated workspace per customer. The agents execute code against the customer's live data and improve per customer through Workspace-RL: one machine runs many customers, each with their own disk and weights, the same code runs everywhere, and no data crosses between customers.

- What Atris is: https://atris.ai/what-is-atris
- Agent-readable source: https://atris.ai/llms.txt

San Francisco. Founded 2025.
